Jar Grid System

Jar Grid for friendly mobile development

Jar is a responsive grid system that will help you create simple responsive sites faster, when creating sites or small examples of mobile views you don't always want to use a super boilerplate or a heavy framework that imports it's own styles. I have been in that situation many times so I created Jar so I can stop spending time choosing and customizing frameworks on my development process, this grid has saved me some time so I decided to share it.

Grid System

Jar's grid is really easy to implement, extreamly fluid and a time saver when dealing with small projects. Check the Readme to understand how to implement Jar.

Jar Utilities

  • Responsive Grid
  • Offset classes
  • Push to left or right by using .lefty .righty
  • Keep columns distribution by usign the class .steady
  • Wrap your content using .container or .container-full
